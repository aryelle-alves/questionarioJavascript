Questão 18 - Módulo de Treinamento

Escreva uma classe Equation que seja capaz de lidar com equações até o segundo grau.

  • Os objetos podem ser instanciados com valores padrões para a = 0, b = 0 e c = 0.
  • Deve ter um método para retornar suas raízes na forma de um array vazio, de um ou de dois elementos.
  • O método também deve descrever os passos para a resolução da equação.
  • Se a, b e c forem todos iguais a 0, ela não deve calcular as raízes e deve retornar uma mensagem de erro.

História Escondida I

Saídas Corretas:

  • Entrada: (0, 2, 6)
    Saída:
    1. 2x + 6 = 0
    2. 2x = -6
    3. x = -6/2
    4. x = -3
    5. [-3]
  • Entrada: ()
    Saída: 'Erro! Nnehum parâmetro informado.'
  • Entrada: (0, 0, 5)
    Saída:
    1. Parametros insuficientes.
      nenhuma raix real.
    2. []
  • Entrada: (1, -4, -5)
    Saída:
    1. ∆ = -4² - 4 * 1 * -5
    2. ∆ = 36
    3. x' = (-(-4) + √36) / 2 * 1
    4. x" = (-(-4) - √36) / 2 * 1
    5. x' = 4 + √36 / 2
    6. x" = 4 - √36 / 2
    7. x' = 4 + 6 / 2
    8. x" = 4 - 6 / 2
    9. x' = 5
    10. x" = -1
    11. [5, -1]
